Web Design Critique

Bad Website View Website →

Consistency: In terms of the colors, the choice of a blue background with black text is not ideal, as at times both the background and text blend in, making it harder for the user to read the content. However, the layout does get the job done, but it could be improved since there are a lot of images, and the layout is inconsistent, as some of the pictures vary in size. There are a lot of colors, and they don't match, making it harder for the user to absorb the content.

Ease of Navigation: Although there is a navigation bar, the font size is very small, and users on their phones will have a really hard time navigating through the website. The links are also really confusing since they are different colors each time, and they are also bright colors. Overall, the navigation is mediocre.

Universal Usability/Accessibility: The text is already small enough and feels congested on a laptop, which makes it even worse on a mobile device. This website is not mobile responsive since the text and layout do not adjust for smaller screens, making navigation and reading difficult on mobile devices.

Error Prevention: The website has bright colors, which cause readers to be distracted. To add on, there is no proper layout, which further confuses the user. Finally, it's also not visually appealing, and it's hard to understand what the website is clearly saying and where to find products.

Good Website View Website →

Consistency: The color layout is visually appealing and matches the overall website (cream, white, pink, pastels). It is also easy to understand what the website is trying to convey due to the minimalistic approach, as its transitions are smooth, and fonts and hover features are consistent.

Ease of Navigation: The navigation bar is visible and bright, which causes the user to be less confused as it easily directs them to the page they may want to go to. The titles are also descriptive, and there are a lot of CTAs (buttons and pop-ups) that take the user directly to the designated link.

Universal Usability/Accessibility: All the features are accessible since there is a lot of space between text, and it is very responsive, as the majority of the text takes you to a different page; they contain hover effects or are fun popups, which all keep the user engaged.

Error Prevention: The website is organized, as it is not overloaded with information, which can minimize errors and guide the user effectively. The overall approach of the website is minimalistic, which makes it easier to find the information the user is looking for.

Team Website Improvement

Improvements Made During Lab: A major change we made to our website in this lab was to have the navigation bar visible on each HTML page, so the user does not have to keep going back, improving efficiency. We also included some hover visuals to make it more appealing, especially on the About Us page, where there are 2 hover options (1 around the body and the other around the View Portfolio button). We were also careful regarding the layout by spacing out text so the reader has an easier time grasping information.
